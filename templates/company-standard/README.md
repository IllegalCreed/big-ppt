# 公司标准汇报模板

## 基本信息

- **模板名称**：公司标准汇报
- **适用场景**：日常汇报、季度总结、年终述职等正式场合
- **配色方案**：红白主题（主色 #d00d14）
- **字体**：微软雅黑（Microsoft YaHei）

## 页面规范

### 封面（cover.md）

- 整体白色背景，内容垂直居中
- 第一行：logo-mark（红色滤镜）+ logo-text（黑色滤镜），右对齐
- 中间：红色渐变背景区域，内含 BIG mark 纹理（拉伸充满，opacity 0.1），白色标题居中
- 第三行：汇报人和时间，左侧黑色粗体
- 占位符：`{{主标题}}`、`{{副标题}}`、`{{汇报人}}`、`{{日期}}`

### 目录（toc.md）

- 整体白色背景
- 左侧：350x350 红色渐变色块，内含 BIG mark 纹理（拉伸充满），十字线分割，下方"目录"+CONTENTS
- 中间：红色竖分割线（3px，与色块间距 20px）
- 右侧：目录条目列表，每项由红底序号方块（60x60）+文字标签组成
- 当前章节 `active`（opacity 1），非当前章节 `inactive`（opacity 0.2）
- 占位符：`{{章节1}}`、`{{章节2}}`、`{{章节3}}`

## 滤镜参考

Logo 图片为白色透明底 PNG，需要通过 CSS 滤镜变色：

- **红色**：`filter: invert(87%) sepia(98%) saturate(4811%) hue-rotate(352deg) brightness(82%) contrast(100%);`
- **黑色**：`filter: brightness(0);`

## 使用注意事项

- 封面必须作为第一页
- 封底必须作为最后一页
- 每个章节前插入一个目录页，高亮当前章节
- 所有页面使用微软雅黑字体
