# 模板索引

本目录包含公司的幻灯片模板套。每套模板由多个页面类型组成，AI 根据用户需求从中选择合适的页面组合成完整演示文稿。

## 可用模板套

| 模板套       | 目录                | 适用场景                     | 页面数量 |
| ------------ | ------------------- | ---------------------------- | -------- |
| 公司标准汇报 | `company-standard/` | 日常汇报、季度总结、年终述职 | 待填充   |

> 更多模板套将陆续添加。

## 页面类型说明

每套模板通常包含以下页面类型：

| 页面类型 | 文件名             | 说明              |
| -------- | ------------------ | ----------------- |
| 封面     | `cover.md`         | 演示文稿开篇页    |
| 封底     | `back-cover.md`    | 演示文稿结尾页    |
| 目录     | `toc.md`           | 展示演示内容概览  |
| 章节分隔 | `section.md`       | 区分不同章节      |
| 通用内容 | `content.md`       | 标准内容展示页    |
| 两栏内容 | `two-col.md`       | 左右两栏布局      |
| 图文混排 | `image-content.md` | 图片+文字混合排版 |
| 数据展示 | `data.md`          | 数据、图表展示    |

## 使用方式

1. 选择合适的模板套
2. 根据内容需求挑选页面类型
3. 将页面组合为完整的 `slides.md`
4. 用 `---` 分隔各页面，添加全局 headmatter
5. 运行 `pnpm dev` 预览

## 新增模板

1. 在 `templates/` 下创建新目录（如 `templates/my-template/`）
2. 创建 `README.md` 描述模板信息和适用场景
3. 添加各页面类型的 `.md` 文件
4. 更新本文件的索引表
